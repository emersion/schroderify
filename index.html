Offline mode

This website can be very useful offline while waiting for a bus/train for example.

Schröderify can be used on mobile phones (e.g. Android). Moreover, it should be available offline automatically in recent browsers (Firefox, Chromium...).

This app is also available on the Firefox Marketplace. You can install Schröderify on any device where Firefox is installed.

About

Schröderify is developped by Emersion. You can visit our Github repository for bugs, ideas and source code.

The project is in beta for the moment, but new features will be added soon, such as declinations, vocabulary and more!
